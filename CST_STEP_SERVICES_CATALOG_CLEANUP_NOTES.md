# CST Services / Pricing Cleanup

- Removed Services from the primary sidebar workflow.
- Global search no longer routes users into a separate service-pricing workflow.
- Services page is now a simple base catalog only.
- Removed active-client counts, customer-specific usage, and customer prices from service catalog cards.
- Client profile Services tab now shows only which services the client has.
- Client-specific prices remain inside Add/Edit Client profile, invoices, and client history logs.
